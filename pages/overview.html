Installation

  • pixi
  • conda
  • pip
pixi workspace channel add munch-group
pixi add vscodenb
conda install -c munch-group vscodenb
pip install vscodenb
Tip

To enable automatic theme detection, you must add the following setting to your vscode settings.json:

"jupyter.themeMatplotlibPlots": true,

You can open settings.json from the command palette (Cmd-Shift-p) by typing “user settings” and select “Preferences: Open User Settings (JSON)”.

from numpy.random import random
import matplotlib.pyplot as plt
import time
plt.rcParams['figure.figsize'] = (5, 3.7)

Less intrusive progress bars

from vscodenb import pqdm, prange

for i in prange(10, persist=True):
    time.sleep(0.1)

SLURM cell magic

Run a single cell as a SLURM job:

%%slurm

s = "Variable values updated in the slurm job are automatically transferred back to the notebook!"
Not running on SLURM cluster.
print(f"It works: {s}")
It works: Variable values updated in the slurm job are automatically transferred back to the notebook!
